# Simulador de Cajero Automático

## Descripción
Este programa simula el funcionamiento básico de un cajero automático. Permite al usuario ingresar su nombre, validar su cuenta y PIN, consultar saldo, depositar, retirar dinero y revisar el historial de movimientos. Está diseñado como proyecto educativo para practicar estructuras de control, clases, funciones, listas y expresiones regulares en Python.

## Instrucciones de uso
1. Ejecuta el programa en un entorno Python .
2. Ingresa tu nombre completo cuando se te solicite.
3. Introduce tu número de cuenta y PIN (ejemplo válido: cuenta `123456`, PIN `7890`).
4. Navega por el menú para realizar operaciones bancarias.
5. Selecciona "Salir" para finalizar la sesión.

## Documentación técnica
- **Lenguaje:** Python 
- **Estructuras utilizadas:**
  - Variables primitivas: `str`, `int`, `float`
  - Variables no primitivas: `list`, `class`
  - Iteraciones: `while`, `for`
  - Decisiones: `if`, `elif`, `else`
  - Funciones propias: `extraer_apellido()`, `validar_credenciales()`
  - Clase definida: `CuentaBancaria`
  - Expresiones regulares: módulo `re` para extraer apellido
  - Búsqueda en historial: filtrado por palabra clave
